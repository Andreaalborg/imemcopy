# IMEM Lifts Norway

Dette er nettsiden for IMEM Lifts Norway - en ledende leverandør av heis- og løfteutstyr i Norge.

## Om prosjektet

Dette repository inneholder:
- Nettside bygget med Next.js og React
- Dokumentasjon og spesifikasjoner
- Markedsmateriell og sertifiseringer

## Teknologi

- **Framework**: Next.js 14+
- **Språk**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Netlify

## Installasjon

1. Klon repository:
   ```bash
   git clone https://github.com/Andreaalborg/imemcopy.git
   cd imemcopy
   ```

2. Installer dependencies:
   ```bash
   cd imem-lifts
   npm install
   ```

3. Start utviklingsserver:
   ```bash
   npm run dev
   ```
